Attribute Description
position: relative The element is positioned relative to the normal position, and is specified with top, right, bottom, left values.
position: absolute It's position is relative to the closest parent element, will move with the rest of the doc when it's scrolled.
position: fixed an element's position is pinned to one part of the web page and will stay there no matter where you scroll.
position: sticky An element can stick to a defined offset position when the user scrolls its parent container.
