CSS Styling Reference

Attribute Description
position: relative The element is positioned relative to the normal position, and is specified with top, right, bottom, left values.
position: absolute It's position is relative to the closest parent element, will move with the rest of the doc when it's scrolled.
position: fixed an element's position is pinned to one part of the web page and will stay there no matter where you scroll.
position: sticky An element can stick to a defined offset position when the user scrolls its parent container.
z-index:(number) Sets how far back/forward an element appears on a page
display: inline the element will take up as little space as possible, can't adjust the width or height
display: block elements take up the width of their container and can adjust teh height
display: inline-block can have a set width and height, can appear next to each other and don't take up the entire container width
float:(left, right) moves elements as far left or right as possible
clear:(left, right) clear the element's left or right
